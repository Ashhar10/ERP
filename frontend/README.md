# ERP Frontend

React-based frontend for the Devine Water ERP Management System.

## Available Scripts

- `npm start` - Runs the app in development mode (http://localhost:3000)
- `npm run build` - Builds the app for production
- `npm test` - Launches the test runner

## Environment Variables

Configure in `.env`:

```env
REACT_APP_SUPABASE_URL=your-supabase-url
REACT_APP_SUPABASE_ANON_KEY=your-supabase-anon-key
REACT_APP_API_URL=http://localhost:5000
```

## Features

- Customer portal with dashboard
- Delivery calendar and tracking
- Financial overview
- Bottle request management
- User profile management
- Glassmorphism UI with dark theme
- Mobile responsive design
